# nwafu-paper

西北农林科技大学本科毕业论文（设计）格式规范技能 for Claude Code。

## 来源

- 格式依据：`关于印发《西北农林科技大学本科毕业论文（设计）撰写规范与编排要求》的通知`（教务〔2026〕3号），参照 GB/T 7713.1-2025 和 GB/T 7714-2025
- 本技能由 **Claude Code**（Anthropic）生成

## 安装

```bash
# 克隆到 Claude Code 的 skills 目录
git clone https://github.com/2p1endid/nwafu-paper.git ~/.claude/skills/nwafu-paper
```

安装后重启 Claude Code，即可通过 `/nwafu-paper` 或提及"西农论文格式"触发。

## 用法

技能会在以下情况自动触发：
- 提到 NWAFU/西农/西北农林科技大学
- 提到本科毕业论文格式、论文撰写规范
- 要求按学校规范格式化 docx 文件
- 显式调用 `/nwafu-paper`

### 示例

```
按西农论文格式要求修改这个docx文件
帮我创建一份符合西北农林科技大学规范的调研报告
检查这篇论文的参考文献格式是否正确
```

## 覆盖范围

| # | 章节 |
|---|------|
| 1 | 页面设置（A4、页边距、页眉页脚） |
| 2 | 字体要求（宋体/黑体 + Times New Roman） |
| 3 | 题名格式（二号黑体居中、单倍行距） |
| 4 | 摘要与关键词格式 |
| 5 | 目录格式 |
| 6 | 正文标题层级（章/一级/二级/三级） |
| 7 | 正文段落（小四宋体、固定20磅、首行缩进） |
| 8 | 图、表、公式编号与排版 |
| 9 | 脚注格式 |
| 10 | 参考文献（著者-出版年制、GB/T 7714） |
| 11 | 附录、致谢、作者简历 |
| 12 | 页眉/页码（罗马+阿拉伯数字） |
| 13 | 装订顺序 |
| 14 | python-docx 实现代码片段 |
| 15 | 字号对照表、DXA换算表 |
